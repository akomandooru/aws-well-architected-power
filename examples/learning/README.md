# Learning Mode Examples

This directory contains examples demonstrating the AWS Well-Architected Power's learning mode, which provides detailed explanations, real-world examples, and anti-patterns to help developers understand Well-Architected best practices.

## What is Learning Mode?

Learning mode is an educational feature that provides:
- **Detailed Explanations**: In-depth descriptions of why best practices matter
- **Real-World Examples**: Practical scenarios showing correct implementations
- **Anti-Patterns**: Common mistakes and why they're problematic
- **Rationale**: The reasoning behind each recommendation
- **AWS Documentation Links**: References to official AWS resources for deeper learning

## How to Use These Examples

1. **Start with a Pillar**: Choose the pillar most relevant to your current work
2. **Read the Explanation**: Understand the "why" behind the best practice
3. **Study the Examples**: See how to implement the pattern correctly
4. **Learn from Anti-Patterns**: Recognize common mistakes to avoid
5. **Follow the Links**: Dive deeper with official AWS documentation

## Interactive Learning

When using the AWS Well-Architected Power with learning mode enabled, Kiro will:
- Provide these detailed explanations automatically during reviews
- Explain the rationale behind each recommendation
- Show relevant examples from this directory
- Link to AWS documentation for further reading
- Offer quiz-style questions to reinforce understanding
